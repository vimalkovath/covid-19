### Build ionic4 with Chart.js

#### Description
`Integrated ChartJS into ionic4 capacitor`

--------------------
#### Clone Project
`npm install`

--------------------
#### Serve Project
`ionic serve -o`
`npm run start`

--------------------
#### Build Project
`npm run build`

--------------------
Build Project Capacitor
`npm run build`
`npx cap copy android | npx cap copy ios`
`npx cap open android | npx cap open ios`
